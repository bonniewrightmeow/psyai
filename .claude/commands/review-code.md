Review the provided code changes or recent commits for:

1. **Research Alignment**
   - Does this support a specific epic/story from the research outline?
   - Which hypothesis does this help validate?
   - What research metrics will this enable?

2. **Logging & Analytics**
   - Are all decisions/actions logged appropriately?
   - Is timing data captured for cognitive load analysis?
   - Are metrics exportable for research analysis?

3. **Privacy & Compliance**
   - Is PII automatically anonymized?
   - Are GDPR requirements met?
   - Is data retention policy followed?

4. **Code Quality**
   - Clear docstrings explaining research purpose?
   - Tests validate research requirements?
   - Performance meets specified targets?

5. **Documentation**
   - Epic/story reference in commit message?
   - Updated relevant documentation?
   - Research implications explained?

Provide specific feedback and suggestions for improvement.
